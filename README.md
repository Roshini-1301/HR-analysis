This project analyzes employee attrition patterns using Python and HR data. The goal is to identify key factors that influence employees leaving an organization and gain insights to improve retention.

**The analysis is performed in a Jupyter Notebook:**
 HR analysis.ipynb

**Dataset Summary**


Records: 14,999 employees


Features: Satisfaction level, projects, salary, department, promotions, tenure, attrition status


Missing values: None

**Tech Stack**

Python

Pandas, NumPy

Matplotlib, Seaborn

Jupyter Notebook

**Key Focus Areas**

Employee attrition trends

Department-wise turnover

Workload vs attrition

Salary and promotion impact

Tenure and satisfaction analysis

**How to Run**

git clone https://github.com/Roshini-1301/HR-analysis.git

cd HR-analysis

jupyter notebook


Open HR analysis.ipynb and run all cells.

**Conclusion**

The analysis highlights that workload imbalance, low salary, lack of promotions, and mid-tenure periods significantly influence employee attrition.
